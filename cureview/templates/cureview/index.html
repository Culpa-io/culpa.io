Frequently Asked Questions

• Who runs this site?

This website is completely open source and run by current Columbia students. If you want to contribute, feel free to open a pull request!

• How do I know it won't go down?

Because Culpa is licensed under the MIT License, anyone can download their own copy of our code at any time and spin up a new version. No single person is in charge of Culpa; if you want to make a change, open a pull request on the GitHub and start the conversation!

• Is it under active development?

Yes! Please join our discord server.

• Additional questions?

Please create an issue on GitHub by clicking this link. Issues allow all of the students who are currently contributing to Culpa to quickly see your message.
